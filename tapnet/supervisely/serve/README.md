# Overview

This app is an integration of TAP-Net model, which is a NN-assisted interactive object tracking model. The TAP-Net model can track point trajectories. It is used to implement the tracking of polygons, points and rectangles on videos.

# How to Run

0. Run the application from Ecosystem

1. Open Video Labeling interface

2. Configure tracking settings

3. Press `Track` button

https://github.com/supervisely-ecosystem/serve-tapnet/assets/115161827/2ef6b5bd-3473-4df6-bf7d-33539377c429

4. After finishing working with the app, stop the app session manually in the `App sessions` tab
